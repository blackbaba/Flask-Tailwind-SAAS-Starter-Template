Welcome to the Flask Tailwind SAAS Starter Template
A comprehensive SAAS starter template with the following features:
  • Basic Flask Application using Blueprints and the Application Factory pattern
  • Logging and Configuration Management for testing, development and production
  • Flask-Login for authentication and authorisation + user and role management
  • Flask-SQLAlchemy and Flask-Migrate for database models and migrations
  • Flask-Mail for sending emails
  • Flask-Assets for JS and CSS assets management
  • Flask-Caching for caching (in-memory/RAM or redis backend)
  • Flask-Limiter for rate limiting
  • Unit, Functional and End-to-End (E2E) testing using some tools
  • TailwindCSS using Tailwind-CLI with Just in Time (JIT) compilation
  • Stripe for payments (coming soon)
  • S3 Integration for file uploads (coming soon)
